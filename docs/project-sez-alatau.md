# Проект: Партнёрство с СЭЗ "Алатау"

## Резюме проекта

| Параметр | Значение |
|---|---|
| Направление | Партнёрство с особой экономической зоной |
| Локация | СЭЗ "Алатау", Алматинская область (п. Алатау, рядом с Алматы) |
| Модель | Win-win: СЭЗ получает развитие территории, Taldau — преференции и льготы |
| Текущий статус | Проработка концепции партнёрства |
| Проектная компания | ТОО "Талдау Алатау" |

## О СЭЗ "Алатау"

### Общая информация

СЭЗ "Парк инновационных технологий" (ПИТ "Алатау") — специальная экономическая зона, созданная для развития информационных технологий, инноваций и высокотехнологичного производства.

- Расположение: п. Алатау, 25 км от центра Алматы
- Площадь: около 164 га
- Управляющая компания: АО "НАТР" (Национальное агентство по технологическому развитию) или назначенный оператор
- Срок действия СЭЗ: до 2035 года (с возможностью продления)

### Преференции для участников СЭЗ

| Льгота | Описание |
|---|---|
| КПН (корпоративный подоходный налог) | 0% (для приоритетных видов деятельности) |
| НДС | 0% на обороты на территории СЭЗ |
| Земельный налог | 0% |
| Имущественный налог | 0% |
| Таможенные пошлины | 0% на импорт оборудования и комплектующих |
| Социальный налог | Пониженная ставка |
| Упрощённый визовый режим | Для иностранных специалистов |

Срок действия преференций — на весь период функционирования СЭЗ (до 2035+).

## Концепция партнёрства

### Модель win-win

**Что получает СЭЗ "Алатау":**
- Инвестиции в развитие территории (строительство, инфраструктура)
- Привлечение новых резидентов и создание рабочих мест
- Улучшение инфраструктуры (коммерческие площади, общественные пространства)
- Рост привлекательности зоны для технологических компаний

**Что получает Taldau Group:**
- Налоговые преференции (0% КПН, НДС, земельный, имущественный)
- Доступ к земельным участкам на льготных условиях
- Статус участника СЭЗ — репутационный капитал
- Возможность реализации девелоперских проектов на территории зоны
- Доступ к программам государственной поддержки

### Направления деятельности на территории СЭЗ

**1. Девелопмент и управление недвижимостью**
- Строительство офисных зданий / коворкингов для IT-компаний
- Коммерческие площади (кафе, магазины, сервисы для резидентов)
- Возможно: жилой комплекс для сотрудников СЭЗ

**2. Технологическая инфраструктура**
- Data-центр или co-location площадка
- Лаборатории / мейкерспейсы
- Конференц-центр / event-пространство

**3. Сервисные функции**
- Управление объектами (facility management)
- Логистика и транспорт (шаттл Алматы — СЭЗ)
- HR-сервисы для резидентов

### Эффект налоговых преференций

При стандартном налогообложении компания платила бы:
- КПН: 20% от прибыли
- НДС: 12% от оборота
- Земельный и имущественный налоги

Экономия от преференций СЭЗ: ориентировочно 60-100 млн тенге/год, что существенно повышает чистую доходность проекта.
